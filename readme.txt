== Description ==
Flexbox is a modern blank theme for your next WordPress project created by developer for developers (not only developers heh). It can be used as a starter template, simple blog, or even woocommerce store. Why it is so cool? It was developed in best development practices: PHP7, OOP, Namespaces, MVC, SCSS and more!
